# Library Management System API

A Django REST Framework API for managing books, authors, borrowers, and borrowing records.

## Features
- Manage books
- Manage authors
- Borrow and return books
- Track borrowers
- Full CRUD API

## Technologies
- Django
- Django REST Framework
- Python

## How to Run

```bash
git clone <your-repo-link>
cd library-management-api
pip install -r requirements.txt
python manage.py runserver
